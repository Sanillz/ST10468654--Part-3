4.4 Project Budget
Even though the Dravanti Media website is a legitimate organization, it was set up as a student project and a budget estimate was made to indicate the cost of running a legitimate website.


Item	Estimated Cost (ZAR)
Domain Name Registration	R250 per year
Web Hosting	R1 200 per year
Website Design	R3 500
Website Development	R5 000
SEO Setup	R2 500
Content Creation	R1 500
Testing and Quality Assurance	R1 000
Website Maintenance	R1 500 per year
Total Estimated Cost	R16 450

Budget Explanation
This budget shows the cost of creating and managing the professional website of Dravanti Media.
Website creation and website design will be the most expensive parts of the process due to the expertise required for both activities. Other expenses include hosting fees, registering a domain name, and SEO implementation and maintenance to keep the website protected and working properly.
In our case, since it is an academic project, there are no financial expenses at all because the website was created using free software and learning materials.
